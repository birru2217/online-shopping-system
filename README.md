# Online Shopping (E-Commerce) Web Application 🛍️🛒

A complete, feature-rich online shopping web application built with PHP, MySQL, and Bootstrap. This platform allows customers to browse products, manage their shopping carts, and place orders, while providing administrators with a dashboard to manage products and orders.

## ✨ Features

### 👤 Customer Features:
- User Registration and Secure Login.
- Browse products by categories (Clothes, Electronics, etc.).
- Multi-language support (English/Amharic/Oromo).
- Interactive Shopping Cart (Add, Update, Remove products).
- Secure checkout process.

### 🔑 Admin Features:
- Admin Dashboard with sales analytics.
- Product Management (Add, Edit, Delete products with image upload).
- Order Management (View and track customer orders).

---

## 🛠️ Technologies Used
- **Backend:** PHP, MySQL (Database connection via `mysqli`)
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
- **Local Server:** WAMP Server / Apache

---

## ⚙️ Installation Guide

1. Clone or download this repository into your local server directory (`wamp64/www/` or `xampp/htdocs/`).
2. Rename the project folder to `onlineshopping`.
3. Open **phpMyAdmin** and create a new database named **`ecommerce`** (Important: The database must be named `ecommerce` to match the configuration).
4. Import the **`onlineshopping_db.sql`** file (which you can download from the link above) into the `ecommerce` database.
5. Open your browser and navigate to: `http://localhost/onlineshopping`.
